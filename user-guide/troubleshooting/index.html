    Troubleshooting

    RoomHub cannot connect to MQTT, even when configured MQTT hostname is correct

    Some ESP32 modules have problems with connection client, probably because of some previously stored data. See GitHub issue for more details https://github.com/espressif/arduino-esp32/issues/180 . Currently the only proved workaround is to change firmware temporarily to call Client.flush();.
